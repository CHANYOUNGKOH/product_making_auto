# 🚀 배포 간단 가이드 (3단계)

## 📋 배포 방법 (3단계만!)

### 1단계: 빌드 (개발 PC에서)

```bash
# 방법 1: 자동화 스크립트 (가장 쉬움) ⭐
빌드_자동화.bat 더블클릭

# 방법 2: 수동 빌드
cd 상품가공프로그램
python build_exe.py
```

**결과**: `dist/상품가공프로그램/` 폴더 생성됨

---

### 2단계: 배포 (다른 PC로 복사)

```
dist/상품가공프로그램/ 폴더 전체를 다른 PC로 복사
```

**포함되는 것:**
- ✅ `상품가공프로그램.exe` (실행 파일)
- ✅ 모든 하위 스크립트 폴더들
- ✅ 필요한 모든 파일들

---

### 3단계: 실행 (다른 PC에서)

1. 복사한 폴더에서 `상품가공프로그램.exe` 더블클릭
2. 끝!

**주의사항:**
- ⚠️ Python 설치 필요 (하위 스크립트 실행용)
- ⚠️ ComfyUI 사용 시: ComfyUI 별도 설치 필요 (이미지 Stage 4)

---

## 🔧 문제 발생 시

### 문제 1: "Python을 찾을 수 없습니다"
**해결**: Python 설치 (다른 PC에)

### 문제 2: Tcl/Tk 오류
**해결**: 
1. 개발 PC에서 `빌드_자동화.bat` 재실행
2. `dist` 폴더 삭제 후 재빌드

### 문제 3: 버튼이 비활성화됨
**해결**: `dist/상품가공프로그램/` 폴더 전체를 복사했는지 확인

---

## 💡 핵심만 기억하기

1. **빌드**: `빌드_자동화.bat` 실행
2. **배포**: `dist/상품가공프로그램/` 폴더 전체 복사
3. **실행**: 다른 PC에서 `.exe` 더블클릭

끝! 🎉
