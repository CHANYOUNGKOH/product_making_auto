================================================================================
                    Stage5 외부 작업 시스템 사용법
================================================================================

[로컬 PC - 보내는 사람]
--------------------------------------------------------------------------------

1. 패키지 생성 (stage5_패키지보내기.py 실행)
   - 엑셀 파일 선택 (I4 또는 I5)
   - 패키지 생성 클릭
   - 이미지는 350x350으로 자동 리사이즈됨 (용량 감소)

2. 생성된 파일:
   stage5_package_작업명/
   ├── manifest.json
   ├── 작업명_T*_I*.xlsx
   ├── images/
   └── README.txt
   + stage5_package_작업명.zip (압축파일)

3. 외부 작업자에게 전송:
   - 최초 1회: Stage5_품질검증_외부용.exe 전송
   - 매번: ZIP 파일만 전송 (구글드라이브, 네이버메일 등)


[외부 PC - 작업자]
--------------------------------------------------------------------------------

1. 최초 1회: Stage5_품질검증_외부용.exe를 원하는 위치에 저장

2. 매번 작업:
   - 받은 ZIP 파일 압축 해제
   - Stage5_품질검증_외부용.exe 실행
   - 압축 해제한 폴더의 엑셀 파일 선택
   - 품질 검증 수행 (1: 누끼만 / 3: 둘다 / 8: 둘다X)
   - 완료 후 "최종 처리" 클릭

3. 작업 완료 후:
   - 폴더 전체를 ZIP으로 압축
   - 로컬 PC로 회신


[로컬 PC - 받는 사람]
--------------------------------------------------------------------------------

1. 받은 ZIP 파일 다운로드

2. stage5_패키지가져오기.py 실행
   - 받은 ZIP 또는 폴더 선택
   - Import 실행 클릭
   - 경로가 자동 복원됨

3. cloudflare_upload_gui.py로 업로드 진행


================================================================================
                              파일 목록
================================================================================

[로컬 PC용]
- stage5_패키지보내기.py    : 패키지 생성 (Export)
- stage5_패키지가져오기.py  : 패키지 가져오기 (Import)
- cloudflare_upload_gui.py : 이미지 업로드

[외부 PC용]
- Stage5_품질검증_외부용.exe : 품질 검증 프로그램 (Python 없이 실행 가능)


================================================================================
                              주의사항
================================================================================

- 외부 PC에는 Python 설치 불필요 (exe만 실행)
- 이미지는 350x350으로 리사이즈되어 전송됨 (리뷰용)
- 원본 이미지는 로컬 PC에 그대로 보존됨
- 패키지 용량: 약 200~450MB (5천~1만개 상품 기준)

================================================================================
