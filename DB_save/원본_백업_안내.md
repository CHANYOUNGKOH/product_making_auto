# 원본 파일 백업 안내

성능 최적화 작업 전에 원본 파일을 백업해야 합니다.

## 백업할 파일

1. `database/db_handler.py` → `database/db_handler_원본_백업.py`
2. `ui/main_window.py` → `ui/main_window_원본_백업.py`

## 백업 방법

### 방법 1: 파일 탐색기에서 수동 복사 (권장)

1. 파일 탐색기를 열고 다음 경로로 이동:
   ```
   C:\Users\kohaz\Desktop\Python\파이썬자동화파일\상품가공프로그램\DB_save
   ```

2. `database` 폴더로 이동
   - `db_handler.py` 파일 선택
   - Ctrl+C (복사)
   - 같은 폴더에서 Ctrl+V (붙여넣기)
   - 복사된 파일 이름을 `db_handler_원본_백업.py`로 변경

3. `ui` 폴더로 이동
   - `main_window.py` 파일 선택
   - Ctrl+C (복사)
   - 같은 폴더에서 Ctrl+V (붙여넣기)
   - 복사된 파일 이름을 `main_window_원본_백업.py`로 변경

### 방법 2: Python 스크립트 실행

터미널에서 다음 명령어를 실행:

```bash
cd "C:\Users\kohaz\Desktop\Python\파이썬자동화파일\상품가공프로그램\DB_save"
python backup_original_files.py
```

## 백업 확인

백업이 완료되면 다음 파일들이 생성되어야 합니다:

- `database/db_handler_원본_백업.py`
- `ui/main_window_원본_백업.py`

백업이 완료되면 성능 최적화 작업을 시작할 수 있습니다.
