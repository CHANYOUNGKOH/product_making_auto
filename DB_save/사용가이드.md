# 사용 가이드

## 🎯 두 가지 도구의 역할

### 1️⃣ data_entry.py - 데이터 입고 도구

**언제 사용하나요?**
- 엑셀 파일에 있는 상품 데이터를 SQLite DB로 저장할 때
- 새로운 상품 데이터를 추가할 때
- 기존 DB에 데이터를 추가할 때

**실행 방법:**
```bash
python data_entry.py
```

**작업 순서:**
1. 📁 엑셀 파일 선택
2. 🏪 마켓 정보 입력 (마켓명, 사업자번호 등)
3. 💾 DB 파일 선택 (새로 만들거나 기존 파일 선택)
4. ▶ DB 변환 시작

**결과:**
- `products.db` 파일 생성 (또는 기존 파일에 데이터 추가)
- 상품 데이터가 SQLite DB에 저장됨

---

### 2️⃣ data_export.py - 데이터 출고 도구

**언제 사용하나요?**
- SQLite DB에 저장된 상품을 마켓에 업로드할 때
- 특정 카테고리의 상품만 업로드할 때
- 여러 마켓에 선택적으로 업로드할 때

**실행 방법:**
```bash
python data_export.py
```

**작업 순서:**
1. 📋 마켓 계정 엑셀 파일 선택 (`Market_id_pw.xlsx`)
2. 💾 DB 파일 선택 (`products.db`)
3. 🔍 카테고리 검색 (예: "원피스")
4. 🏪 마켓/스토어 선택 (체크박스)
5. ▶ 시작

**결과:**
- 선택한 마켓에 상품이 업로드됨
- 업로드 이력이 DB에 기록됨

---

## 🔄 전체 작업 흐름

```
┌─────────────┐
│  엑셀 파일   │
│ (상품 데이터) │
└──────┬──────┘
       │
       ▼
┌─────────────────┐
│  data_entry.py  │  ← 데이터 입고
│ (엑셀 → SQLite)  │
└──────┬──────────┘
       │
       ▼
┌─────────────┐
│ products.db │  ← SQLite 데이터베이스
│  (SQLite)    │
└──────┬──────┘
       │
       ▼
┌─────────────────┐
│ data_export.py  │  ← 데이터 출고
│ (SQLite → 마켓)  │
└──────┬──────────┘
       │
       ▼
┌─────────────┐
│   마켓 API   │
│ (쿠팡, 네이버) │
└─────────────┘
```

## 💡 자주 묻는 질문

### Q1: 두 파일을 모두 실행해야 하나요?

**A:** 네, 순서대로 실행하세요.
1. 먼저 `data_entry.py`로 엑셀 데이터를 DB에 저장
2. 그 다음 `data_export.py`로 DB의 데이터를 마켓에 업로드

### Q2: 엑셀 파일을 수정했어요. 다시 실행해야 하나요?

**A:** 
- **상품 데이터 엑셀**: `data_entry.py`를 다시 실행하여 DB에 반영
- **마켓 계정 엑셀** (`Market_id_pw.xlsx`): `data_export.py`에서 "🔄 다시 로드" 버튼 클릭

### Q3: 같은 DB 파일을 계속 사용해도 되나요?

**A:** 네, 같은 `products.db` 파일을 계속 사용하면 됩니다. 
- 새 데이터를 추가할 때는 기존 DB 파일을 선택하면 자동으로 추가됩니다.

### Q4: 마켓 계정 정보는 어디에 저장되나요?

**A:** 
- `Market_id_pw.xlsx` 파일에 저장 (시트별로 마켓 구분)
- 프로그램에서 선택한 경로는 `config_settings.json`에 저장되어 다음 실행 시 자동으로 로드됩니다.
