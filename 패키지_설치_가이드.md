# 📦 패키지 설치 가이드

## 🚀 빠른 설치 (자동)

### 방법 1: 배치 파일 사용 (가장 쉬움) ⭐

```bash
패키지_설치.bat 더블클릭
```

끝! 자동으로 모든 패키지가 설치됩니다.

---

## 📋 수동 설치 방법

### 방법 2: 명령어 사용

```bash
# 1. 프로젝트 폴더로 이동
cd 상품가공프로그램

# 2. 패키지 설치
pip install -r requirements.txt
```

---

## ⚠️ 주의사항

### GPU 사용 시 (배경 제거 등)

배경 제거 기능에서 GPU를 사용하려면:

```bash
# CUDA가 설치된 경우 (NVIDIA GPU)
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118

# CPU만 사용하는 경우
pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
```

**참고**: `requirements.txt`에는 torch가 주석 처리되어 있습니다. 필요 시 수동으로 설치하세요.

## 🔧 문제 해결

### 문제 1: "pip를 찾을 수 없습니다"

**해결**: Python이 제대로 설치되지 않았습니다.
1. Python 재설치
2. 설치 시 "Add Python to PATH" 체크 확인

### 문제 2: "권한 오류"

**해결**: 관리자 권한으로 실행
```bash
# 명령 프롬프트를 관리자 권한으로 실행 후
pip install -r requirements.txt
```

### 문제 3: 특정 패키지 설치 실패

**해결**: 개별 설치
```bash
pip install 패키지명
```
